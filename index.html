Sobre Mim

Olá! Me chamo Gerard Tercio, sou desenvolvedor web de 20 anos. Gosto de criar sites e aplicações modernas, funcionais e fáceis de usar. Estou sempre aprendendo novas tecnologias e buscando projetos que me desafiem tanto no design quanto na programação.

Projetos

Aqui estão alguns dos projetos em que trabalhei — cada um deles um passo na minha jornada como desenvolvedor. De sites responsivos a aplicativos web dinâmicos, esses projetos refletem minha paixão por código limpo, design moderno e resolução de problemas do mundo real. Estou sempre experimentando, aprendendo e construindo.

Fanta Landing Page

Este projeto é uma landing page inspirada na marca Fanta, desenvolvida com HTML, CSS e a biblioteca de animações GSAP. A proposta foi criar uma interface visualmente atrativa, divertida e moderna, destacando elementos como frutas, cores vibrantes e efeitos visuais suaves.

Formulário de registro de usuário

Um formulário de cadastro de usuário limpo e estruturado, desenvolvido com HTML e CSS. Este projeto apresenta uma ampla variedade de tipos de entrada, incluindo texto, e-mail, senha (com validação), botões de opção, caixas de seleção, seletor de data, upload de arquivo e muito mais. Também inclui um layout responsivo com seções separadas para imagens e campos de formulário, demonstrando atenção à organização do layout e à experiência do usuário. Este projeto destaca minhas habilidades de front-end em design de formulários, HTML semântico e validação básica de formulários.

Expense Refund Form

Um formulário de solicitação de reembolso responsivo e fácil de usar, criado com HTML, CSS e JavaScript. Este projeto permite que os usuários enviem detalhes de despesas, como categoria e valor, e exibe um resumo em tempo real de todas as solicitações. Inclui componentes de interface limpos, como menus suspensos, validação de entrada e rastreamento dinâmico de despesas (via JavaScript). O design segue um estilo profissional e moderno, usando Google Fonts e um layout claro com formulário e resumo lado a lado. Este projeto demonstra minhas habilidades em criar interfaces funcionais e estéticas baseadas em formulários.
